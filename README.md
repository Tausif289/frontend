📱 React Native App – Easy Start Guide

This project is built using React Native with the React Native CLI.

🚀 Step 0: Before You Start

Make sure your system is ready (Node.js, Android Studio/Xcode installed).

👉 Follow: https://reactnative.dev/docs/set-up-your-environment

▶️ Step 1: Start the App Server (Metro)

Metro is the tool that runs your app in development.

Run this in your project folder:

npm start
# OR
yarn start
📱 Step 2: Run the App

Open a new terminal (keep Metro running) and run:

🤖 Android
npm run android
# OR
yarn android
🍎 iOS (Mac only)

First time setup:

bundle install
bundle exec pod install

Then run:

npm run ios
# OR
yarn ios
✏️ Step 3: Edit Your App
Open App.tsx
Make changes
Save the file

✅ Your app updates automatically (Fast Refresh)

🔄 Reload App (if needed)
Android: Press R twice OR Ctrl + M → Reload
iOS: Press R in simulator
🎉 Done!

Your app is now running and ready to develop 🚀

Build your own app 🔥
